Legend

( 1) discontinued,
( 2) approximate,
( 3) coat d'arms match the OLD citadel colours so there might be some difference. Shining gold has the most noticeable difference,
( 4) P3 Paints did not really match very closely with the old GW range, but a detailed explanation of each paint can be found here: http://www.brushthralls.com/pre-painting-prep/color-theory-10p3.html
( 5) GW's new range matches with the old range are the matches claimed by GW. However, many of the paints do not match very closely at all and some mixing will be required to get close to the original colours. If you are trying to match old GW with new GW, you'd be better just getting Vallejo game color or coat d'arms for the closest match )
(color name) after testing does not match GW color (really far)
( 6) 98-99% colour matches, colour difference is so minute it's very difficult to tell the difference when painting models side by side
( 7) 3rd-5th Edition Replicas, same 98-99% matching criteria
